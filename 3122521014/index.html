BABAGAN KULO


Ing papan puniki kulo badhe njlentrehaken babagan kulo. Tepanaken, nami kulo Muhammad Rizki Darmawan. Kulo miyos ing Pati 14, febuari 2004. Umur kulo puniki sampun 18 taun. kulo puniki kuliah ing Politeknik Elektronika Surabaya ing kampus Lamongan kulo mendhet jurusan teknik informatika ing kuliah puniki.
Ing Kampus Puniki kulo mboten gadah prestasi anangin kulo gadah tuisan ing SMA. Ing SMA kulo remen banget kalian sastra. kulo nderek lomba nulis anangin mboten wonten ingkang gadah juara. Ing SMA kulo ugi nderek organisasi Yaiku PMR. Ing PMR kulo dados Kadiv PSDM. Ing Kuliah Kulo ngerjaken project sing dados tugas kuliah.
